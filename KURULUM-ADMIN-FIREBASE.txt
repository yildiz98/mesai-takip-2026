MESAİ TAKİP 2026 - Firebase Admin Kurulumu

Bu sürüm SmartApart Firebase projesine bağlanır.

1) Firebase Authentication > Users bölümünde şu kullanıcıyı oluştur:
   E-posta: admin@mesaitakip.app
   Şifre: Poh1234

2) Firestore Rules kısmında şu kurallar yayınlanmış olmalı:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /smartapart/{document=**} { allow read, write: if request.auth != null; }
    match /mesaiTakip/{document=**} { allow read, write: if request.auth != null; }
    match /users/{document=**} { allow read, write: if request.auth != null; }
    match /logs/{document=**} { allow read, write: if request.auth != null; }
    match /settings/{document=**} { allow read, write: if request.auth != null; }
  }
}

3) GitHub'a zip içindeki dosyaları eski dosyaların üstüne yükle.

Giriş:
Kullanıcı adı: admin
Şifre: Poh1234

Normal kullanıcılar uygulama içinden Kayıt Ol ile hesap oluşturabilir.
Kullanıcılar e-posta görmez; sistem arkada kullaniciadi@mesaitakip.app formatı kullanır.

Not: Admin panelindeki Sil işlemi kullanıcıyı uygulama içinde pasif/engelli yapar. Firebase Authentication'dan tamamen silmek için Firebase Console > Authentication > Users ekranı kullanılır.
