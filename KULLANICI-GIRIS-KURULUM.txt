MESAİ TAKİP - KULLANICI ADI / ŞİFRELİ GİRİŞ KURULUMU

1) Firebase Console'da Authentication > Sign-in method > Email/Password seçeneğini aktif edin.
2) Firestore Database oluşturun.
3) Authentication > Users > Add user ile admin hesabını oluşturun:
   Email: yildiz.21.98@gmail.com
   Password: kendi belirleyeceğiniz şifre
4) Firebase web config bilgilerini firebase-config.js dosyasına yapıştırın.
5) Firestore Rules bölümüne firestore.rules dosyasındaki kuralları yapıştırıp Publish edin.
6) GitHub'a tüm dosyaları yükleyin.

KULLANICI GİRİŞ MANTIĞI
- Personel ekranda sadece Kullanıcı Adı + Şifre görür.
- Sistem arkada kullanıcı adını otomatik olarak kullaniciadi@mesai.local formatına çevirir.
- Admin maili: yildiz.21.98@gmail.com
- Admin hesabı kayıt ekranından değil Firebase Authentication > Add user bölümünden oluşturulur.
- Admin bu mail ile giriş yaptığında otomatik Admin olur.

ADMIN PANELİ
- Toplam kayıtlı kullanıcı sayısı
- Son giriş tarihleri
- Kullanıcı engelleme
- Kullanıcıyı silinmiş işaretleme
- Admin / Personel yetkisi verme

ÖNEMLİ NOT
Tarayıcı içinden Firebase Authentication kullanıcısını tamamen silmek güvenli şekilde yapılamaz. Paneldeki Sil işlemi kullanıcıyı uygulama içinde silinmiş/engelli işaretler ve girişini engeller. Firebase Authentication listesinden tamamen kaldırmak için Firebase Console veya Cloud Function gerekir.
